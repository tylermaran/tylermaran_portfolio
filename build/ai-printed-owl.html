GPT can 3D Print!

It's clear that LLM can do a whole lot more than search text and write code. Microsoft even demo'd ChatGPT as an interface for robotics. So how well can it do 3D designs?

OpenSCAD

OpenSCAD.

GPT refuses to just dump an STL file for me, but is more than happy to generate OpenSCAD code. If you haven't used it before, OpenSCAD is a FOSS CAD software designed for people who think that JavaScript is the optimal tool for creating 3D models.
